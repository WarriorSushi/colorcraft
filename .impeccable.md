# ColorCraft Design Context

## Target Audience
- UI/UX designers building design systems and brand palettes
- Frontend developers needing accessible color combinations
- Design-conscious indie hackers choosing brand colors
- Ages 22-40, design tools daily, care about quality and speed

## Use Cases
- Generate harmonious palettes for a new project in seconds
- Check WCAG accessibility compliance before committing to colors
- Preview how palettes look to people with color vision deficiency
- Export palette directly as CSS variables, Tailwind config, or SCSS
- Save and compare multiple palette iterations

## Brand Personality / Tone
- **Precise and confident** — a sharp tool, not a toy
- **Artisan craft** — feels like a professional color lab, not a random generator
- **Fast and responsive** — zero friction, instant feedback
- **Opinionated** — suggests the best option, not all options
- **Inspired by:** Figma's precision, Linear's density, Coolors.co's speed

## Visual Direction
- **Aesthetic:** Industrial design tool — dense information, sharp edges, no wasted space
- **Colors:** Deep black (#0a0a0b) base, the palette itself IS the color — minimal chrome
- **Typography:** Space Grotesk for headings (geometric precision), DM Sans for body (clean, modern), JetBrains Mono for hex values
- **Corners:** Small to medium radius (rounded-lg max), sharp and precise
- **Borders:** Crisp zinc-800, no blur/glass effects — this is a tool, not a dashboard
- **Effects:** Minimal — focus on the colors themselves, not UI decoration
- **Motion:** Snappy 100-150ms transitions, no bounce, no elastic
- **Layout:** Left-aligned, asymmetric, dense — maximize information per pixel
- **NEVER USE:** violet, purple, fuchsia, gradient text, glassmorphism, bounce animations, emoji as icons

## Differentiator
The palette strip IS the hero — each color swatch is tall and prominent with all info visible at a glance. The tool disappears behind the colors. Spacebar to regenerate instantly (like Coolors). Color blindness preview is not buried — it's one tab away.

## Technical Stack
- Next.js 16 + TypeScript + Tailwind CSS
- Static export for GitHub Pages
- 100% client-side, zero backend
- Lucide React icons
- Mobile-first responsive
